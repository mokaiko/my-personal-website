# 私の個人ホームページ

https://mokaiko.vercel.app/

私の個人ホームページを展示しています。このウェブサイトリンクには基本情報が含まれています。

## 機能特徴

### バージョン1

- ✅ 多言語対応 (日本語、簡体字中国語、英語)
- ✅ モバイルフレンドリーなデザイン

## 技術スタック

- JavaScript
- Tailwind CSS
- Vercel デプロイ

## 開発

```bash
# 依存パッケージのインストール
npm install

# 開発モード
npm run dev

# ビルド
npm run build

# 本番モード
npm start
```
